==============================================================================
<h2>Usage</h2>
Run UniversalEssentialLib target
Go to EssentialLib -> build -> lib.a -> copy & import to new project
Find "Build Settings/Header Search Paths" add "$SOURCE_ROOT/"
In ViewController
```
#import "EssentialLib/file.h"
```
